# Generador de Informes HTML y PDF

Este proyecto permite crear informes de manera automática utilizando plantillas HTML y datos dinámicos. Produce reportes consistentes y profesionales, con posibilidad de incluir gráficos, tablas y exportación a PDF usando Playwright. El entorno y la ejecución se administran con **uv** para simplificar el flujo de trabajo.

---

## ✨ Características

- Renderizado de plantillas HTML con **Jinja2**.  
- Manejo de datos con **pandas**.  
- Soporte para gráficos creados con **matplotlib**.  
- Lectura y escritura de archivos Excel mediante **openpyxl**.  
- Exportación opcional a PDF con **Playwright**.  
- Gestión y ejecución del proyecto usando **uv**.
